HyperText Markup Language (HTML) is the standard markup language[a] for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

HTML History

HTML was created by Tim Berners-Lee in 1991 while he was working at CERN. The first version of HTML was quite simple compared to the current standard, with only a handful of elements, including <html>, <head>, <title>, <body>, <h1> to <h6>, <p>, and <a>.

HTML version timeline

HTML 2

November 24, 1995

HTML 2.0 was published as RFC 1866. Supplemental RFCs added capabilities:
  1. November 25, 1995: RFC 1867 (form-based file upload)
  2. May 1996: RFC 1942 (tables)
  3. August 1996: RFC 1980 (client-side image maps)
  4. January 1997: RFC 2070 (internationalization)

December 18, 1997

HTML 4.0[19] was published as a W3C Recommendation. It offers three variations:
  • Strict, in which deprecated elements are forbidden
  • Transitional, in which deprecated elements are allowed
  • Frameset, in which mostly only frame related elements are allowed.

Sekarang Belajar Tata Bahasanya HTML yaitu Semantic Markup

Penjelasan sederhananya, jadi kalau semantic markup itu tujuan pembuatan sesuai dengan codingnya. Semantic markup di HTML adalah cara menulis kode HTML dengan tag yang punya arti/makna sesuai dengan isi kontennya, bukan cuma untuk gaya (style). Misalnya: Kalau bikin judul, pakai <h1> ... </h1>, bukan <div> lalu dikasih CSS besar-besar.

  • Kalau bikin paragraf, pakai <p> ... </p>, bukan <span> ... </span>.
  • Kalau bikin navigasi menu, pakai <nav> ... </nav>.
  • Kalau bikin artikel, pakai <article> ... </article>.
  • Kalau bikin bagian utama halaman, pakai <main> ... </main>.

Dengan semantic markup, kode HTML jadi lebih jelas, mudah dimengerti, dan lebih baik untuk SEO serta aksesibilitas. Ada beberapa contoh coding semantic diantaranya :

  • <header> ... </header> : untuk bagian atas halaman (header)
  • <nav> ... </nav> : untuk navigasi menu
  • <main> ... </main> : untuk konten utama halaman
  • <article> ... </article> : untuk artikel atau postingan
  • <section> ... </section> : untuk bagian atau seksi tertentu
  • <aside> ... </aside> : untuk konten sampingan atau tambahan
  • <footer> ... </footer> : untuk bagian bawah halaman (footer)
